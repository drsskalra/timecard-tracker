## Key Points

1. **Offline-First**: App works without internet, syncs when available
2. **Conflict-Safe**: User controls conflict resolution
3. **Secure**: Multiple layers of security
4. **Fast**: Only syncs changed data
5. **Reliable**: Uses Microsoft's infrastructure
6. **Private**: Your data, your cloud
7. **Free**: No cost for typical usage

## Performance Characteristics

- **Initial Sync**: < 1 second (empty data)
- **Regular Sync**: < 500ms (typical week data)
- **Large History**: 1-2 seconds (years of data)
- **Token Refresh**: Automatic, silent
- **Bandwidth**: < 10 KB per sync (typical)
- **Storage**: < 1 MB per year of timecards
