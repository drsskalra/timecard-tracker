## Usage

1. Visit the website: [https://drsskalra.github.io/timecard-tracker/](https://drsskalra.github.io/timecard-tracker/)
2. Enter your project information (Project Name, Project Number, Task Name)
3. Click "Start Timer" to begin tracking
4. Click "Stop Timer" when done
5. Export your timecard to Excel at the end of the week
